Bug found and fixed during smoke test:

The first live tmux run against /tmp/chat-log-live.sqlite failed because EvalTool decoded replsession.ExecutionReport.Result, which is a display preview and may be truncated with an ellipsis. The model saw errors such as:

- eval_js result was not valid JSON: invalid character 'â' after object key:value pair
- eval_js result was not valid JSON: unexpected end of JSON input

Fix:

- The eval wrapper now stores the exact JSON result in globalThis.__chat_eval_last_json.
- After replapi.App.Evaluate returns, EvalTool reads that exact string with replapi.App.WithRuntime.
- The fixed run against /tmp/chat-log-live-fixed.sqlite passed.
